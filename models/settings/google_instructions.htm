1. Go to Google Developers Console and select your project, otherwise create one using Create Project.
  2. Once the project is created, goto API Manager from the navigation bar and click Enable API. Search for the Google+ API and add it.
  3. Now in the side menu select the Credentials and click on Create Credentials -> OAuth Client ID. Choose the type as Web Application and give it a name such as “For website login”. Set the Authorized redirect URIs to your web app’s page URL where the Google Login component is to be added (such as /login) and click create. It will show you the Client ID and Client Secret which should be copied into the Google tab fields in the backend social login settings page.
